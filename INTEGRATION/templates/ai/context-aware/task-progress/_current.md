## 📝 How to Use This File

**At start of session:**
1. Add your task (copy template above)
2. Mark status as "In Progress 🔵"
3. List subtasks

**During work:**
1. Check off subtasks as completed
2. Update status if blocked
3. Document any findings

**At end of session:**
1. Mark complete or in-progress
2. Move to `completed/[date].md` if done
3. Commit with message: `✅ Task complete: [name]`

**For next agent:**
1. Read this file
2. Knows exactly what's done and what's not
3. Can continue smoothly
